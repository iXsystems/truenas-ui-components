Stepped Wizard Pattern

For linear workflows

The Stepped Wizard pattern combines the tn-dialog and tn-stepper components to create guided, multi-step workflows. This pattern uses the stepper in linear mode to ensure users complete steps in the correct order. For non-linear wizards where users can jump between steps freely, use the Tabbed Wizard Pattern instead.
