# Theming in TrueNAS UI Library

This library supports multiple themes. You can switch themes using the **Storybook toolbar**.

## Available Themes
- **iX Dark** (default)
- **iX Blue**
- **Dracula**
- **Nord**
- **Paper**
- **Solarized Dark**
- **Midnight**
- **High Contrast**

## Usage in Components
Use CSS variables to apply theme styles:

```scss
.my-component {
  background-color: var(--bg1);
  color: var(--fg1);
  border: 1px solid var(--lines);
}
```

## Theme Variables Reference

Each theme defines the following CSS custom properties:

### Core Colors
- `--primary`: Primary brand color
- `--primary-txt`: Text color for primary backgrounds
- `--accent`: Accent color for highlights
- `--topbar`: Top navigation bar background
- `--topbar-txt`: Top navigation bar text

### Background Colors
- `--bg1`: Main background color
- `--bg2`: Secondary background color (cards, panels)
- `--alt-bg1`: Alternative background 1
- `--alt-bg2`: Alternative background 2

### Text Colors
- `--fg1`: Primary text color
- `--fg2`: Secondary text color
- `--alt-fg1`: Alternative text color 1
- `--alt-fg2`: Alternative text color 2

### UI Elements
- `--lines`: Border and divider color
- `--btn-default-bg`: Default button background
- `--btn-default-txt`: Default button text

### Status Colors
- `--red`: Error/danger color
- `--green`: Success color
- `--yellow`: Warning color
- `--blue`: Info color
- `--orange`: Secondary warning
- `--cyan`: Info accent

## Implementation Notes

### Component Integration
Components should use the `--lines` variable for borders and dividers:

```scss
.component {
  border: 1px solid var(--lines);
  border-bottom: 1px solid var(--lines);
}
```

### Fallback Values
When using CSS variables, always provide fallback values for better compatibility:

```scss
.component {
  background-color: var(--bg1, #1E1E1E);
  color: var(--fg1, #ffffff);
  border: 1px solid var(--lines, #383838);
}
```
